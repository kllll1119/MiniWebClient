# MiniWebClient
最简单的Web桌面开发框架（Windows）

==============================
========== Why MiniWebClient? ==========
1、使前端开发桌面程序更加便捷，减少桌面程序的学习成本
2、修改一个配置前端即可进行桌面程序的逻辑开发，几乎零成本
3、提供C++端对前端的功能支持，使前端亦可操作计算机底层
